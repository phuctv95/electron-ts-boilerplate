# Electron TypeScript Boilerplate

A boilterplate of Electron that:

- Uses TypeScript.
- Uses pnpm.
- Uses esbuild.
- Supports hot reload for renderer.
- Has constants for channels.

Main files:

```
src/
    main/
        main.ts
    renderer/
        (Svelte files for renderer here)
    preloader.ts
```

To start devlelopment, run `pnpm run dev`.
